Job Opportunity: Back-End Developer (Laravel)

We are looking for a skilled Back-End Developer to join our team. This role is an excellent opportunity for individuals passionate about building efficient, scalable, and secure web applications.

Job Responsibilities:

  • Develop, maintain, and optimize back-end services using Laravel.
  • Design and implement RESTful APIs and database structures.
  • Ensure application security, scalability, and performance.
  • Troubleshoot, debug, and optimize application logic.
  • Collaborate with front-end developers and other team members.
  • Write clean, maintainable, and well-documented code.

Required Qualifications & Experience:

  1. Proven experience as a Back-End Developer with Laravel.
  2. Strong understanding of PHP, MySQL, and database design principles.
  3. Experience with API development and integration (RESTful APIs).
  4. Proficiency in Git for version control.

Preferred Skills:

  • Experience with cloud services (AWS, DigitalOcean).
  • Knowledge of caching techniques (Redis, Memcached).
  • Understanding of front-end technologies (Vue.js, React, or Angular) is a plus.
